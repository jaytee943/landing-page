# landing-page

Therapy & coaching landing page, built with [Astro](https://astro.build) + TypeScript, deployed
to GitHub Pages via GitHub Actions.

Live at: https://jaytee943.github.io/landing-page/

See [NOTES.md](./NOTES.md) for project background and [CLAUDE.md](./CLAUDE.md) for working rules.

## Commands

All commands are run from the root of the project, from a terminal:

| Command             | Action                                     |
| :------------------ | :----------------------------------------- |
| `pnpm install`      | Install dependencies                       |
| `pnpm dev`          | Start local dev server at `localhost:4321` |
| `pnpm build`        | Build production site to `./dist/`         |
| `pnpm preview`      | Preview the build locally                  |
| `pnpm check`        | Type-check the project                     |
| `pnpm lint`         | Lint with ESLint                           |
| `pnpm lint:fix`     | Lint and auto-fix                          |
| `pnpm format`       | Format with Prettier                       |
| `pnpm format:check` | Check formatting without writing           |
